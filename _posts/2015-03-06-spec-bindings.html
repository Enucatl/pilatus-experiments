--- layout: post title: Spec bindings ---
  • Send commands from spec to the detector server with the sock_put function.
  • The sock_get command is used to wait for the commands to run on the server and display the result if appropriate.
  • The macros are set up so that the camserver saves the image in raw binary format: a C-style ordered 619x487 sequence of int32.
  • After the image is saved, it is cropped with the dectris-pilatus-roi script. The original image is overwritten.
